1. Program Description

Program Name: CrewLedger Receipt Tracking System

Operated By: Roofing & Renovations of Florida LLC

CrewLedger is an automated employee expense receipt tracking system that works via SMS text messaging. Authorized employees submit receipt photographs by texting them to the CrewLedger business phone number. The system reads the receipt, sends a confirmation message for the employee to verify, and stores the data for company accounting and reporting.

4. How the Service Works

Using CrewLedger involves the following steps:

  • First Contact: The first time you text the CrewLedger number, you introduce yourself. The system registers your name and phone number.
  • Submitting a Receipt: Text a photo of your receipt along with the project name. Example: [photo] Project Sparrow
  • Confirmation: The system reads the receipt and sends you a summary (vendor, amount, items, project). Reply YES to confirm or NO to flag it for review.
  • Missed Receipts: If you did not receive a receipt for a purchase, text the CrewLedger number and the system will walk you through logging it manually.
